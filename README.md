 - 매주 일요일 오후 9시 스터디 진행

 - 스터디가 끝나고 다음 스터디에 발표할 Item을 정합니다.
 - 매 스터디 전날 자정까지 발표 자료를 PR로 올립니다.
 - 발표가 끝나고 질문 시간을 갖습니다.
 - 질문과 그에 대한 답은 Issues를 통해서 남깁니다.
 - 발표나 자료에 대해서 궁금한 내용을 남겨주세요.
